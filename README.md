#### Como iniciar um projeto do Framework usando Grunt:
---
```shell
git clone http://github.com/luishc/boilerplate-framework-php.git . && \
git clone http://github.com/luishc/FRAMEWORK-PHP.git FRAMEWORK/ && \
rm -rf .git && \
rm -rf README.md && \
rm -rf FRAMEWORK/.git && \
rm -rf FRAMEWORK/.gitignore && \
git init && \
git add . && \
git status && \
git commit -m "Init"
```
